# Zecser AI System

AI-powered Job Portal with Resume Parsing, ATS, Screening and Scoring.

## Tech Stack
Python, FastAPI, NLP, Machine Learning

## Folder Structure
- parsers/
- ats_engine/
- screening_ai/
- interview_ai/
- scoring/
- utils/
- tests/

## Setup
1. Create virtual environment
2. Install dependencies
3. Run main.py
